Create an event

Two ways to create an event within the dashboard:

  1. Create an event on multiple assets: on /assets page, select one or multiple assets, from bulk actions dropdown choose create event.
  2. Create one event on one asset: on specific asset page, click Add event button

Create event form

Options:

  • Basic info:
    • Event type: ie. ambrosus.asset.sold
    • Name
    • Description
    • Access level: 0 (event is public) or 1+ (event is private, *isn't displayed on amb.to)
  • Event documents (one or more)
  • Any other custom information, formated as:
    • Key-value: Origin: ...
    • Group information: Product information: size: ... weight: ... expiration date: ... brand: ....
  • Event identifiers: GTIN, GLN, QR... (one or more)
  • Event location
  • Asset location
