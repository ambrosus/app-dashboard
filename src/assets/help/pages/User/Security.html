Security

In settings > security tab, besides password change form, user can see the list of all the devices current account is logged in.

On each login user session is created and stored in the browser, meaning no one can pretend to be a specific user in order to hack users account. Using sessions is a secure way to handle login process.

Each session describes:

  • Device type
  • Operating system
  • Browser

On logout session on that device is deleted.

If user forgot to logout from different devices, for more security user can logout of all those devices inside of the dashboard.

To logout of the current session (current browser), there's a logout button in the top right menu (by click on users name or picture).
