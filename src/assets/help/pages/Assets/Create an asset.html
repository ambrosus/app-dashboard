Create an asset

Two ways to create an asset within the dashboard:

  1. Regular form
  2. JSON form

To create an asset, of course, user doesn't have to know any of the details explained in what are assets, rather it's easily done using regular or JSON form, which creates an asset and info event describing the asset automatically.

On /assets page, click on a Create asset button on the right side in table's header.

Regular form

Options

  • Basic info: assetType, name and description
  • Asset images (one or more)
  • Asset identifiers: GTIN, GLN, QR... (one or more)
  • Any other custom information, formated as:
    • Key-value: Origin: ...
    • Group information: Product information: size: ... weight: ... expiration date: ... brand: ....

Regular form process:

  • Creates an asset.
  • Creates an event of type ambrosus.asset.info, which holds all the descriptive information about the asset, and attaches the event to the asset.

JSON form

In JSON form, user can just upload or insert a template of an asset with multiple events attached to it, and this form will create a new asset with the same events and information in them attached to the asset.

Where the creator of the asset and its events is set as the current logged in user, using the form.
