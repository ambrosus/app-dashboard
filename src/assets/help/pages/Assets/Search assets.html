Search assets

On /assets page, user can use the search bar to search for asset based on:

  • name
  • createdBy (address of asset creator)
  • asset identifiers GLN: 123, GSIN: 1234, DATAMATRIX: 12345
  • event identifiers (same as above)
  • type: ie. sold

Search gets all the events and displays unique list of assets that have searched events attached to them.
