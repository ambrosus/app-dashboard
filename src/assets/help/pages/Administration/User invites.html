User invites

The only way for users to create accounts within the dashboard, they need to be first invited by dashboard's owner or admin.

  • In administration page > invite tab, owner or admin can send one or multiple invites.
  • For each invite, inviter can set:
    • AccessLevel - Hermes Node accessLevel giving user ability to create assets/events with different level of privacy.
    • Role - Each role has array of specific permissions, ie. admin can manage users (within the dashboard) and create assets and events, whereas normal user can only view and create assets and events.
  • Additional to the email for each invite, inviter can set a message to be sent with the invites.
  • Once sent, sent/not yet used invites, can be managed inside of the invites tab, where owner or admin can revoke one or more invites.

The process of receiving an invite is explained in creating an account question.
