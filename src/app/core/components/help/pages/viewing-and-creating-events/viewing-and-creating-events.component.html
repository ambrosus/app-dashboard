Viewing and creating events

You can view the events and add new ones by navigating to the specific page of the asset of interest.

To create a new event, click on “Create Event”. You will have two ways to add event data:

  1. The user-friendly input form (default). Use it to simply fill out all the necessary fields.
  2. Another, a rather technical way, is to use a JSON-formatted text (read more).

Once the fields are filled out, click “Create”. If all the information you entered is valid, you will see a progress bar indicating the upload status.
